In some circumstances it can be annoying to have content on the page jumping around when switching between tabbed content that varies in length. In these cases, you can add the class tabs-fixed-height to your .fs-tabbed element; this will match the height of all tabbed content to the tallest one.

Deep Linking Tabs

Tab deep linking allows you to open a specific tab on page load by passing in the cooresponding hashtag in the URL. For this to work you need to add class tabs-deep-link to your .fs-tabbed element. And then when you link to your page containing the tabbed content, you'll add a hash to your URL formatted like #tab_{id of tab to open}.

For example, the following set of tabs has three content panes with the ID's, alpha, beta, and theta. If we wanted to open the beta pane on page load, we would link to tabbed.html#tab_beta.
